# StepFi-Web

React TypeScript web application for the StepFi BNPL protocol on Stellar.

## Stack

Vite, React 18, TypeScript, Tailwind CSS, Zustand, TanStack Query, Freighter API

## Getting Started

```bash
npm install
npm run dev
```
